# Vagrant Ansible

Provision Ubuntu 14.04 Vagrant machine with Ansible. Although roles on their own can be used as well.

## Roles

 - Image Magick
 - Nginx
 - NodeJS
 - PHP7
 - PostgreSQL
 - Ruby
 - Ubuntu Packages
 - Vagrant Specifics
 - Git Customization
 - AWS CLI
 - MySQL. Based on [geerlingguy Ansible MySQL Role](https://github.com/geerlingguy/ansible-role-mysql)
